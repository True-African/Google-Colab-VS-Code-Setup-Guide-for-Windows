Run Google Colab notebooks directly inside VS Code on Windows with free GPU/TPU access. A complete guide to installing the official Google Colab extension for VS Code, connecting to Colab runtimes, and leveraging cloud compute from your local editor.

## Features

- ✅ Install the official Google Colab extension (by Google)
- ✅ Install Jupyter extension prerequisite
- ✅ Connect VS Code to your Google account
- ✅ Run notebooks with CPU, GPU (T4), or TPU runtimes
- ✅ Switch between runtime types seamlessly
- ✅ Properly disconnect and clean up resources

## Why Use This?

| Feature | Benefit |
